+---------------------------+--------------------------------------------+--------------------------------------------------------------+
| Feature                   | BeanFactory                                | ApplicationContext                                           |
+---------------------------+--------------------------------------------+--------------------------------------------------------------+
| Definition                | Basic IoC container for managing beans     | Advanced IoC container built on top of BeanFactory           |
+---------------------------+--------------------------------------------+--------------------------------------------------------------+
| Interface Type            | Root interface                             | Sub-interface of BeanFactory                                 |
+---------------------------+--------------------------------------------+--------------------------------------------------------------+
| Bean Initialization       | Lazy Loading (creates beans when needed)   | Eager Loading (creates singleton beans at startup by default)|
+---------------------------+--------------------------------------------+--------------------------------------------------------------+
| Startup Performance       | Faster startup                             | Slightly slower startup                                      |
+---------------------------+--------------------------------------------+--------------------------------------------------------------+
| Runtime Performance       | First bean access may be slower            | Faster bean access since beans are pre-created               |
+---------------------------+--------------------------------------------+--------------------------------------------------------------+
| Dependency Injection      | Supported                                  | Supported                                                    |
+---------------------------+--------------------------------------------+--------------------------------------------------------------+
| Bean Scope Support        | Basic bean scopes                          | All bean scopes including web scopes                         |
+---------------------------+--------------------------------------------+--------------------------------------------------------------+
| Event Handling            | Not Supported                              | Supported                                                    |
+---------------------------+--------------------------------------------+--------------------------------------------------------------+
| Internationalization      | Not Supported                              | Supported                                                    |
| (i18n)                    |                                            |                                                              |
+---------------------------+--------------------------------------------+--------------------------------------------------------------+
| Annotation Support        | Limited                                    | Full support (@Component, @Autowired, etc.)                  |
+---------------------------+--------------------------------------------+--------------------------------------------------------------+
| AOP Integration           | Limited                                    | Full support                                                 |
+---------------------------+--------------------------------------------+--------------------------------------------------------------+
| Resource Handling         | Basic                                      | Advanced                                                     |
+---------------------------+--------------------------------------------+--------------------------------------------------------------+
| Typical Use Case          | Lightweight applications                   | Enterprise & modern Spring applications                      |
+---------------------------+--------------------------------------------+--------------------------------------------------------------+
| Common Implementations    | XmlBeanFactory (Deprecated)                | ClassPathXmlApplicationContext                               |
|                           |                                            | AnnotationConfigApplicationContext                           |
|                           |                                            | WebApplicationContext                                        |
+---------------------------+--------------------------------------------+--------------------------------------------------------------+
| Recommended Usage         | Rarely used today                          | Preferred in modern Spring applications                      |
+---------------------------+--------------------------------------------+--------------------------------------------------------------+
